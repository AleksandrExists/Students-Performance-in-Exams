# Анализ успеваемости студентов

Анализ датасета об успеваемости студентов с kaggle

## Данные
- Источник: [Students Performance in Exams](https://www.kaggle.com/datasets/spscientist/students-performance-in-exams)
- Файл: `StudentsPerformance.csv`
- Размер: 1000 строк, 8 столбцов

## Что сделано:
1. Загрузка датасета
2. Определение переменных для анализа
3. Получение уникальных значений категорий
4. Построение диаграмм распределения результатов экзаменов
5. Построение диаграмм с долей студентов, сдавших экзамен
6. Построение boxplot с распределением результатов экзаменов по категориям
